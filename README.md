<a href="#">ProPresenter Mac</a> operator experience is the application's most important design
consideration. Unlike PowerPoint or Keynote — where presentations are prepared in advance and
advanced linearly slide-by-slide — ProPresenter is used by a dedicated operator during a live
event who jumps between songs, displays scripture on request, triggers video content, and
manages multiple outputs simultaneously.

The operator is the invisible technical production role at every modern church service and
many live events — the person whose decisions about what appears on screen directly affect
the congregation or audience experience. <a href="#">ProPresenter Mac</a> is designed for
this role specifically, with quick-access controls, keyboard shortcuts for common operations,

## The Church Production Workflow

### Sunday Morning Preparation

Before a Sunday service, the <a href="#">ProPresenter Mac</a> setup session covers:

**Song arrangement confirmation**: The set list from the worship director is loaded into
ProPresenter. Each song is checked for correct lyrics, current arrangement order, and
appropriate background visual.

**Scripture preparation**: Scripture passages for the sermon are pre-loaded with the
correct translation, reference formatting, and slide breaks at appropriate verse boundaries.

**Announcement slides**: Pre-service and post-service announcement slides are arranged in
the announcement loop channel.

**Video content**: Sermon intro videos, announcement videos, or supporting media clips are
added to the appropriate locations in the presentation flow.

**Stage display configuration**: The stage display for the worship team is configured with
the correct layout for the worship leader — typically current lyric, next lyric, a clock,
and any MIDI click track information.

### Live Service Operation

During the service, the <a href="#">ProPresenter Mac</a> operator workflow:

**Song presentation**: For each worship song, the operator advances through slides with arrow
keys or mouse clicks. ProPresenter displays the correct lyrics on house screens and the
countdown preview on the operator's screen simultaneously.

**Scripture display**: When the pastor calls out a scripture reference, the operator types
the reference in ProPresenter's scripture lookup and sends the verse to screens while the
pastor is speaking.

**Video triggering**: Pre-service videos and announcement videos are triggered at the correct
moment. ProPresenter's video playback controls allow pause, play, and precise timing.

**Announcement transitions**: The announcement loop switches automatically or is manually
transitioned as the service moves between segments.

## ProPresenter 6 Migration Guide

Many churches and venues remain on <a href="#">ProPresenter 6 download for Mac</a> and are
considering migration to ProPresenter 7. Understanding the migration process:

### Library Compatibility

ProPresenter 7 can import ProPresenter 6 song and presentation libraries. The import process
converts the library format — which takes time for large libraries — but preserves the content.
Some formatting may require review after import if the templates differ between versions.

### Template Recreation

ProPresenter 6 and 7 use different template systems. Custom slide templates built for version
6 do not automatically migrate to the version 7 template format. Templates need to be
recreated or adapted in ProPresenter 7's template system.

### Parallel Operation

During migration, running both ProPresenter 6 and 7 on the same Mac allows testing ProPresenter 7
with the new library while maintaining the reliable ProPresenter 6 installation for live services
until the team is comfortable with the new version.

## Stage Display Design

<a href="#">ProPresenter Mac</a> stage display is an independent output channel distinct from
the main house screens, designed for the confidence monitors facing performers and worship
leaders:

**Clock**: A clock displaying current time and optional service time elapsed helps the worship
team maintain timing without looking at personal devices during the service.

**Current and next lyric**: The current lyric in large text and the next lyric in smaller text
below allow the worship leader to read the current lyrics while preparing for the next line.

**MIDI sync**: BPM and click track information synchronized from MIDI input helps bands
maintain tempo with the click track without visible drum kit monitoring.

**Custom layouts**: Multiple stage display layouts are configured for different service segments —
a worship segment layout shows lyrics; a spoken word segment layout shows notes or sermon
outline; a prayer layout shows prayer content without lyrics.
